# Changelog

## 0.1.0

- Initial release.
- **Open Mindmap Preview** command (Command Palette, editor-title button,
  `Ctrl/Cmd+K M`) opens a live mindmap beside the active Markdown editor.
- Renders headings, bullets, `Q:` questions, `- [ ]` / `- [x]` tasks, and
  `[label](url)` links.
- Fit / Expand / Collapse controls; wheel-zoom, drag-pan, click-to-collapse.
- Preview follows the focused Markdown file and updates as you type.
